# Accounts Service

[![Build Status](https://github.com/dwikanugraha13/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)](https://github.com/dwikanugraha13/devops-capstone-project/actions)

This repository contains the Accounts Service microservice, part of the IBM DevOps Capstone Project.

## Overview
A RESTful microservice to manage customer accounts for an e-commerce platform.

## Tech Stack
- Python / Flask
- PostgreSQL
- Docker / Kubernetes
- Tekton CI/CD
